Политика конфиденциальности Inline Sync Core

Обновлено: 29 октября 2025 г.

1. Общие положения

Настоящая Политика конфиденциальности описывает, как сервис Inline Sync Core (далее — «Сервис») обрабатывает и защищает персональные данные пользователей.

Используя Сервис, вы соглашаетесь с условиями данной Политики.

2. Какие данные мы собираем

  • Ваш Telegram ID или e-mail (для регистрации и авторизации);
  • Технические данные: IP-адрес, тип устройства, язык браузера, время запросов к API;
  • Адреса криптокошельков, добавленные пользователем для мониторинга.

Мы не запрашиваем и не храним приватные ключи, seed-фразы и иные чувствительные данные доступа к кошелькам.

3. Как мы используем данные

  • Для предоставления функциональности Сервиса (уведомления, API, статистика);
  • Для связи с пользователем (уведомления о транзакциях, технические оповещения);
  • Для улучшения качества работы Сервиса и анализа нагрузки на систему.

4. Хранение и защита данных

Данные пользователей хранятся на защищённых серверах. Доступ к ним имеют только уполномоченные сотрудники с необходимым уровнем прав.

Все соединения с Сервисом защищены с использованием протокола HTTPS и шифрования данных.

5. Передача данных третьим лицам

Мы не передаём ваши персональные данные третьим лицам, за исключением случаев, когда это необходимо для:

  • выполнения требований законодательства;
  • предоставления доступа к сторонним сервисам по вашему запросу (например, Telegram Bot API).

6. Права пользователя

Вы имеете право:

  • запросить копию ваших данных;
  • потребовать их удаление из системы;
  • отозвать согласие на обработку данных.

Для этого свяжитесь с нами по контактам ниже.

7. Изменения в Политике

Inline Sync Core оставляет за собой право обновлять настоящую Политику. Актуальная версия всегда доступна на странице https://inline-sc.pro/privacy.
