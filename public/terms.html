1. Общие положения

Настоящее Пользовательское соглашение регулирует отношения между пользователем (далее — Пользователь) и сервисом Inline Sync Core (далее — Сервис), предоставляющим доступ к API, SDK и веб-интерфейсу для мониторинга блокчейн-транзакций.

Используя Сервис, Пользователь подтверждает, что ознакомился и согласен с условиями настоящего Соглашения.

Сервис предоставляет технический доступ к API и инструментам мониторинга данных из публичных блокчейнов (Ethereum, Cosmos, TON, Bitcoin и др.).

Inline Sync Core предоставляет только технологические инструменты для обработки и получения публичных данных блокчейна.

3. Регистрация и доступ

Для использования API требуется регистрация через Telegram-бота или электронную почту. После регистрации Пользователю предоставляется уникальный API-ключ. Пользователь несет ответственность за его сохранность.

Сервис имеет право временно или навсегда ограничить доступ Пользователя в случае нарушения настоящего Соглашения.

Inline Sync Core предоставляет платные и бесплатные тарифные планы.

Стоимость, лимиты и условия тарифов указаны на сайте или в Telegram-боте @inline_sync_core_bot.

Оплата осуществляется авансом за выбранный период. При неуплате доступ к API может быть ограничен. Средства, уплаченные за уже активированные тарифы, не возвращаются.

Inline Sync Core оставляет за собой право ограничить скорость запросов и приостановить доступ при превышении лимитов.
